Course Description

Algorithms and principles involved in machine learning with applications to various engineering domains; fundamentals of representing uncertainty, learning from data, supervised learning, unsupervised learning, and learning theory; design and analysis of machine learning systems; design and implementation of a technical project applied to real-world datasets.

Course Philosophy

This course embraces a “Learn ML by Building” philosophy:

  • Fundamentals First: Core concepts and unifying principles form the foundation
  • Scaffolded Learning: Progressive complexity with provided code skeletons
  • AI Portfolio: Build a collection of projects as mini research papers
  • Practical Impact: Bridge theory and application through hands-on implementation

Course Learning Objectives

Upon successful completion of this course, students will be able to:

  1. Analyze and contrast broad classes of machine learning algorithms (supervised vs. unsupervised)
  2. Describe and apply fundamental concepts of learning from data (maximum likelihood estimation, maximum a posteriori, overfitting vs underfitting, regularization, cross-validation)
  3. Design and implement supervised learning algorithms for regression and classification
  4. Apply unsupervised learning algorithms for clustering and dimensionality reduction
  5. Build and train neural networks using modern deep learning frameworks
  6. Implement basic reinforcement learning agents
  7. Evaluate ML systems for performance, fairness, and societal impact
  8. Communicate complex ML concepts clearly through written reports and presentations

Course Assessment

Project 1: Foundations 15% KNN and fundamental principles
Project 2: Neural Networks 15% Deep learning architectures & applications
Project 3: Reinforcement Learning 15% Advanced topics (RL/GenAI)
Capstone Project 45% Self-directed research project
Educational Content 5% “Explain ML to a Kid” videos (2 required)
Participation 5% Piazza contributions, peer engagement
Flex/Adjustment up to 5% Reserved for bonus opportunities

Project Specifications

  • Format: Mini research papers following conference paper structure
  • Submission: PDF report via Canvas, code via GitHub
  • Collaboration: Individual work required for Projects 1-3
  • Capstone: Teams of 2-3 allowed with contribution statements

Late Policy

  • Three (3) free late days for the entire semester (use strategically)
  • After exhausting free days: 10% penalty per day
  • No late submissions accepted for final capstone project
  • Medical/emergency extensions require documentation through Dean of Students
